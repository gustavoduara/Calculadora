# Calculadora

![](https://i.imgur.com/0ccZIEe.gif)

## Descrição

Calculadora simples que permite realizar as operações básicas da matemática

## Funcionalidades

- **Operações básicas**: Adição, subtração, multiplicação e divisão
- **Suporte a decimais**: Permite realizar operações com até duas casas decimais
- **Validação de operações**: Evita que o usuário entre com operações inválidas
- **Tratamento de divisão por zero**: Exibe uma mensagem de erro caso o usuário tente dividir por zero
- **Tabuada**: Exibe a tabuada do número informado pelo usuário
- **Histórico de operações**: Capaz de armazenar um histórico de operações realizadas

## Como ultilizar

1. Clone o repositório ou baixe o codigo fonte
2. Abra o terminal ou o prompt de comando e navegue até a pasta raiz
3. Ultilize o comando a baixo para restaurar as dependencias do projeto



```
dotnet restore
```

4. Em seguida, complete a solução ultilizando o comando:
```
``
dotnet build
```

`5. Para executar o projeto, utilize o comando:

```
dotnet run --project .\Calculadora\Calculadora.csproj
```

6. Para executar o arquivo compilado navegue até a pasta ./Calculadora/bin/Debug/net5.0 e execute o arquivo Calculadora.exe
